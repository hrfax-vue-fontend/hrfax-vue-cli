# hrfax-vue-fontend

hrfax-vue-fontend is a cli tool for frontend develop

## Install

    $ [sudo]npm install -g hrfax-vue-cli

## Usage 
    # 查看模板
    $ [sudo]hrfax-vue list
    # 初始化项目
    $ [sudo]hrfax-vue init
